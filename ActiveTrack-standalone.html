Required for Google Calendar sync. How to get it?

  1. Go to console.cloud.google.com
  2. Create a project (or select an existing one)
  3. Enable the Google Calendar API
  4. Go to APIs & Services → Credentials
  5. Create an OAuth 2.0 Client ID (Web application)
  6. Add null as origin (for file:// access) or host on GitHub Pages for full support
  7. Copy the Client ID and paste it above
